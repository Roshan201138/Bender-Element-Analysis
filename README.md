# Bender Element Signal Interpretation App

## Overview
This application processes bender element test data to estimate:
- Shear wave velocity (Vs)
- Small-strain shear modulus (Gmax)

Two interpretation methods:
- Peak-to-peak
- Normalized cross-correlation

## Features
- CSV, Excel, TXT support
- Single & batch processing
- Dual-axis plots (V / mV)
- Export: Excel, CSV, PNG

## How to Use
1. Upload file(s)
2. Select columns
3. Set units and parameters
4. Run analysis
5. Review plots

## Notes
Signals are treated as discrete . Cross-correlation is normalized.
Always visually verify results.
